## Generating JavaScript documentation
Install typedoc globally using:

	npm install typedoc --global

In a terminal, navigate to your RadJav/html5 directory, then enter:

	npm install

Then navigate to this directory and do one of the following:

Windows

	build-js-docs.bat

Linux/Mac

	./build-js-docs.sh

## Generating C++ documentation
Install doxygen from http://www.doxygen.org/

You can use Doxywizard and click File -> Open, then navigate to your RadJav/docs directory and select Doxyfile.
